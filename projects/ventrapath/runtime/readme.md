# VentraPath Runtime Pack

This folder contains the executable-ready prompt/config layer for the initial VentraPath agent team.

## Contents

- `agents.yaml` — agent registry, scope, latency budgets, prompt paths
- `phase0-routing.yaml` — Phase 0 routing policy and hot-path rules
- `prompts/` — one prompt per agent

## Important rule

This prompt pack is **build-ready**, not **production-approved**.

Production approval only happens after the Phase 0 latency benchmark passes.
