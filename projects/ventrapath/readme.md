# VentraPath

This folder is the durable home for VentraPath work.

## Rules

- Chat is scratch space. Files are the source of truth.
- Update these docs as decisions are made.
- Commit at every meaningful milestone.
- Before risky changes, make a checkpoint commit first.

## Core files

- `agents.md` — agent roster, roles, owners, prompts, interfaces
- `decisions.md` — durable decisions and why they were made
- `todo.md` — current build queue and blockers
- `recovery.md` — how to checkpoint, recover, and safely change things
