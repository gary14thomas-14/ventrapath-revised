# VentraPath 30-Case Benchmark Batches

Run the 30-case matrix in 5 mixed-difficulty batches of 6 cases each.

## Batch structure

Each batch contains:
- 2 simple cases
- 2 medium cases
- 2 messy cases

This keeps pass-rate tracking more honest than clearing all the easy stuff first.
